# ⚡ WiFi Security Scanner

A Python-based cybersecurity tool to analyze WiFi networks and detect potential threats.

---

## 🚀 Features

- 📡 WiFi Network Scanning
- 🔐 Encryption Detection (WPA2, WPA3, Open)
- 🔑 Password Strength Analysis
- 👀 Evil Twin Detection
- ⚠️ ARP Spoofing Detection (Scapy)
- 📶 Signal Strength Monitoring
- 📊 Latency & Packet Loss Analysis
- 🚨 Risk Classification (Green / Yellow / Red)
- 🔥 Live Web Dashboard (Flask)

---

## 🛠 Tech Stack

- Python
- Flask
- Scapy
- HTML/CSS
- JavaScript
